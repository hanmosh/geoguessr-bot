# geoguessr-bot

this repo creates a basic cnn model that can identify countries by their image, positioning it to play the [geoguessr](https://www.geoguessr.com/) game

## How to run

### EDA (Exploratory Data Analysis) 

located in the `/eda` folder

this section is where we explored our [kaggle dataset](https://www.kaggle.com/datasets/ubitquitin/geolocation-geoguessr-images-50k/data) and is in the format of a .ipynd notebook which you can run by uploading it into colab


### Model

you can run our model by cloning this repo into colab and running the individual data splitting, train, and eval python files 

further instructions in this notebook: https://colab.research.google.com/drive/1GPhenwrd7GKXcq-YyOjn8e_nUU8WZXCU?usp=sharing
